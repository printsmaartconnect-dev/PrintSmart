# PrintSmart — Setup Guide

## 1) Install dependencies
```bash
npm install
```

## 2) Start dev server
```bash
npm run dev
```

## 3) Build for production
```bash
npm run build
npm run start
```

## Notes
- This project uses Next.js App Router under the `app/` directory.
- Tailwind CSS is configured via `tailwind.config.js` and `postcss.config.js`.
